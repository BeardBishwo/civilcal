TO START DEBUGGING (3 SIMPLE STEPS)
================================================================================

STEP 1: Install Xdebug Extension for PHP
   Download: https://xdebug.org/download
   File: php_xdebug-3.3.2-8.3-vs16-x86_64.dll
   Copy to: C:\laragon\bin\php\php-8.3.16\ext\php_xdebug.dll

STEP 2: Configure php.ini
   Open: C:\laragon\bin\php\php-8.3.16\php.ini
   Add at end:

   [Xdebug]
   zend_extension=php_xdebug.dll
   xdebug.mode=debug,develop
   xdebug.start_with_request=yes
   xdebug.client_port=9003
   xdebug.client_host=127.0.0.1
   xdebug.idekey=VSCODE

STEP 3: Restart Laragon

HOW TO USE DEBUG PANEL
================================================================================

1. Press Ctrl+Shift+D (opens Debug panel)

2. Install "PHP Debug" extension in VS Code:
   Ctrl+Shift+X ‚Üí Search "PHP Debug" ‚Üí Install

3. Set breakpoints:
   - Click left of line numbers (red dot appears)
   - Try: public/index.php line 11

4. Select configuration from dropdown:
   "Listen for Xdebug" (for browser)
   OR
   "Debug Test Script" (for CLI)

5. Press F5 or click green play button

6. For browser debugging:
   - Visit http://localhost/Bishwo_Calculator/
   - VS Code pauses at breakpoint

7. Debug controls:
   F5        - Continue
   F10       - Step Over
   F11       - Step Into
   Shift+F11 - Step Out
   Shift+F5  - Stop
