API ENDPOINT USAGE
================================================================================

GET /api/profile.php
  Description: Retrieve user profile data
  Auth: Session-based (requires login)
  Response: JSON object with user data
  
PUT /api/profile.php  
  Description: Update user profile
  Auth: Session-based (requires login)
  Body: JSON with fields to update (first_name, last_name, company, phone)
  Response: JSON object with updated fields

Example:
  1. Login: POST /api/login.php
  2. Get Profile: GET /api/profile.php  
  3. Update: PUT /api/profile.php with JSON body
