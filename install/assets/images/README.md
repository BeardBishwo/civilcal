# Bishwo Calculator - Installation Assets

This directory contains the installation wizard assets.

## Files

- `css/install.css` - Professional installation styling
- `js/install.js` - Installation wizard JavaScript functionality  
- `images/logo.png` - Installation wizard logo (placeholder)

## Usage

These assets are automatically loaded by the installation wizard at `/install/`.

## Customization

- Modify `install.css` to change the visual appearance
- Update `install.js` to add new interactive features
- Replace the logo placeholder with your actual brand logo

---

**Note:** These files are part of the installation system and may be removed during the security cleanup process after installation is complete.
